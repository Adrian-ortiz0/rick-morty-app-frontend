# Rick & Morty App

Aplicación SPA desarrollada con **Angular 19** que consume la **Rick&MortyAPI**, implementando paginación, búsqueda y una UI moderna usando **Tailwind CSS + DaisyUI**.

---

##  Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

- **Node.js** (versión LTS recomendada)
- **Angular CLI**  
  ```bash
  npm install -g @angular/cli
  
git clone https://github.com/Adrian-ortiz0/rick-morty-app-frontend

cd rick-morty-app-frontend

npm install

ng serve -o lo abré inmediatamente

Este proyecto utiliza Tailwind CSS como framework de estilos.
